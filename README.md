# 🚀 Lancement rapide

1. **Se placer dans un dossier de travail**

```bash
cd /chemin/vers/ton/dossier
```

2. **Cloner les deux repositories**

```bash
git clone https://github.com/Challenge-48HYnov/DEVChallenge48H.git
git clone https://github.com/Challenge-48HYnov/DATA.git
```

3. **Entrer dans le projet principal**

```bash
cd DEVChallenge48H
```

4. **Lancer les services avec Docker**

```bash
docker compose up -d
```
